# 桌面端冒烟清单

> [English version →](desktop-smoke-checklist_EN.md)

本清单用于每次 Windows 桌面端发版前的人工冒烟验证。它只覆盖真实桌面能力，不替代自动化测试。

## 基础启动

- [ ] 双击启动 Windows 可执行文件，主窗口居中显示。
- [ ] 首次启动时出现三步配置向导。
- [ ] 完成首次配置后，今日入账、工作进度、已工作时间和今日预计正常显示。
- [ ] 关闭后重新打开，配置与首次启动完成状态保持一致。

## 设置保存

- [ ] 修改薪资模式、金额、工作日、上下班时间后，主看板立即更新。
- [ ] 修改主题、金额变换方式、窗口置顶状态后，关闭并重启仍保持。
- [ ] 输入临时无效薪资配置时，界面出现提示，主题和窗口偏好仍可保存。

## 托盘和单实例

- [ ] 最小化或关闭窗口后，托盘图标仍存在。
- [ ] 左键点击托盘图标可以重新显示主窗口。
- [ ] 托盘菜单可以打开主界面、打开设置、切换迷你模式、切换置顶和退出。
- [ ] 应用运行中再次启动同一可执行文件，不会打开第二个主窗口，已有窗口被唤起并聚焦。

## 迷你悬浮

- [ ] 双击主金额进入迷你悬浮模式。
- [ ] 迷你悬浮窗口可拖动，双击或按 Enter / Space 可以恢复主窗口。
- [ ] 右键迷你悬浮窗口打开透明度面板。
- [ ] 调整透明度后，迷你悬浮窗口透明度即时变化并在重启后保持。

## 桌面环境

- [ ] 深浅色切换后，窗口四角、边框和主面板没有明显闪白或残影。
- [ ] 系统休眠后恢复，今日入账不会倒退或出现异常瞬跳。
- [ ] 在多显示器环境中移动主窗口和迷你悬浮窗口，重启后位置和尺寸恢复合理。
- [ ] 高 DPI 缩放下，主窗口、设置面板、迷你悬浮和透明度面板文字不重叠。

## Web Preview 边界

- [ ] Web Preview 中不显示托盘、置顶、开机自启动等桌面专属能力。
- [ ] Web Preview 修改配置只影响浏览器本机 localStorage，不影响 Windows 桌面端配置。

## 记录方式

发版前请记录测试版本、Windows 版本、显示器数量、DPI 缩放和失败项截图。若任一关键项失败，应先修复或明确标记为已知问题，再继续发版。
