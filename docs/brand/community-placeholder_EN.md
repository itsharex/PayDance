# Community Placeholder Assets

> [中文版 →](community-placeholder.md)

This area is reserved for generic placeholder assets that forks and community derivatives can use as references or replacements.

If placeholder assets are added later, they should be dedicated to the public domain under [CC0 1.0 Universal](https://creativecommons.org/publicdomain/zero/1.0/) so the community can use them without inheriting official PayDance branding.

## What belongs here

- Generic application icons not derived from official PayDance branding
- Replaceable placeholder logos
- Neutral social preview images

## What does not belong here

- Modified versions of official PayDance logos, icons, or marketing artwork
- Any official brand assets listed in [`official.md`](official.md)
- Visual assets that could confuse users about the origin of a distribution

## Adding assets

Community members are welcome to contribute high-quality, original placeholder assets that can be clearly licensed under CC0. Submit a PR with the asset source and licensing notes.
