## 使用问题

想确认”这个功能是不是这样用”，可以先在 [GitHub Issues](https://github.com/MasterBao66/PayDance/issues) 搜索关键词。没有相关讨论时，提交一个普通 Issue，写清使用场景和期望结果即可。

## Bug 反馈

请优先使用仓库中的 Bug 反馈表单，尽量提供：

- 应用版本，如 `v0.9.3`
- 系统环境，如 Windows 11 23H2
- 稳定复现步骤或大致触发场景
- 期望结果与实际结果
- 截图、录屏或报错片段

请勿公开粘贴薪资数据、配置文件或私钥。

## 功能建议

薪跳的核心是”桌面实时工资看板”：轻量、本地优先、低打扰。提交功能建议时，请说明它解决什么具体场景，以及为什么服务这个核心体验。平台适配建议请同时说明目标系统、验证方式和你愿意维护的范围。

快捷键、提醒通知、历史图表、打卡考勤、账号同步等方向目前不在产品边界内。接近这些范围的建议仍可讨论，但需要解释为什么不会让产品变重。

## 安全漏洞

请勿通过公开 Issue 报告安全漏洞。请按 [SECURITY.md](../.github/SECURITY.md) 中的方式私下报告。
