# PayDance v0.5.11 Settings And Night Shift Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship v0.5.11 with a calmer four-section settings center, overnight work shift support, cleaner author/repository attribution, and no changes to shortcut, reminder, timeline, or mini floating behavior.

**Architecture:** Keep the existing Vue 3/Tauri 2 app shape. Extend `src/lib/salary.ts` so time windows can normalize across midnight, then refactor `src/components/SettingsPanel.vue` to present the same persisted settings through four clear groups. Version and documentation changes stay mechanical and isolated.

**Tech Stack:** Vue 3, TypeScript, Vitest, Tauri 2, Rust, GitHub Actions.
