# Community Placeholder Assets

This directory is reserved for generic placeholder assets that forks and
community derivatives can use as starting points.

Placeholder assets (if added) are dedicated to the public domain under
[CC0 1.0 Universal](https://creativecommons.org/publicdomain/zero/1.0/).

## What belongs here

- Generic application icons (not derived from PayDance branding)
- Placeholder logos
- Neutral social preview images

## What does NOT belong here

- Modified versions of official PayDance logos or icons
- Brand assets from the `official/` directory
- Any artwork that could confuse users about the origin of a distribution

## Adding assets

Community members are welcome to contribute high-quality, original placeholder
assets under CC0. Submit a PR with your contribution.
