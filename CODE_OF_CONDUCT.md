# 行为准则

> [English version →](docs/CODE_OF_CONDUCT_EN.md)

薪跳 PayDance 欢迎围绕产品、代码、文档和发布流程展开具体、友善、有依据的讨论。项目目前由个人维护，请尽量聚焦问题、说明背景，并给出可验证的信息。

## 我们希望看到

- 用尊重、清晰的语言交流，允许不同观点存在。
- 提建议时说明使用场景、影响范围和验证方式。
- 做技术反馈时尽量提供复现步骤、截图、日志或代码依据。
- 讨论保持在 PayDance 的产品边界内：本地优先、轻量、低打扰的桌面实时工资看板。
- 涉及商业授权、OEM、白标或品牌使用时，主动说明相关背景。

## 我们不接受

- 人身攻击、嘲讽、威胁、骚扰，或持续跑题。
- 公开他人隐私，或要求贡献者透露与贡献无关的薪资、雇主、身份、联系方式。
- 冒充 PayDance 官方版本、维护者或发布渠道。
- 使用带有性暗示、歧视性或侮辱性的文字和图片。
- 明知超出项目边界，仍反复推动明显增加维护负担的方向。

## 遇到问题怎么办

如果遇到行为准则问题，可以通过 [获取帮助](docs/SUPPORT.md) 或 [安全策略](.github/SECURITY.md) 中的渠道联系维护者。

维护者会根据情况删除评论、关闭讨论、拒绝贡献，或限制相关账号参与。处理目标不是惩罚，而是让项目讨论保持安全、聚焦、可继续。
